Если этот блок не исчезает за секунду, страница открыта как файл с диска (file://). Корневой index.html в этой папке рассчитан на сервер разработки Vite, а не на двойной щелчок по файлу.

После npm run build откройте из этой же папки проекта файл open-from-folder.html (двойной щелчок) — так удобнее, чем лезть в dist. Либо dist/index.html.

  • Разработка: в папке проекта выполните npm run dev и откройте адрес из терминала.
  • Запуск с диска: npm run build, затем этот же корень при открытии файла перенаправит на dist/index.html, либо команда npm run open-dist (Windows).
